# 服务端部署说明

本文说明如何启动交互式缠论图表服务，并提供外部浏览器访问入口。

## 功能

- 首页默认加载 `SH000001` 上证指数。
- 支持快捷选择：
  - `SH000001`
  - `000001.SZ`
  - `000905.SH`
  - `SH000852`
  - `SH000300`
  - `SH000016`
  - `688111.SH`
  - `002475.SZ`
- 支持手动输入股票代码查询。
- 图表支持滚轮缩放、拖拽平移、双击十字星、悬停查看 OHLC。
- 服务端按请求实时拉取行情并生成 HTML 图表。
- 默认使用通达信 `mootdx` 数据源，页面也保留东方财富数据源作为可选项。

## 启动命令

本地访问：

```bash
/opt/homebrew/bin/python3.11 web_server.py --host 127.0.0.1 --port 8000
```

服务端外部访问：

```bash
/opt/homebrew/bin/python3.11 web_server.py --host 0.0.0.0 --port 8000
```

如果服务端 Python 路径不同，可替换为：

```bash
python3 web_server.py --host 0.0.0.0 --port 8000
```

## 访问地址

入口页：

```text
http://服务器IP:8000/
```

本机测试地址：

```text
http://127.0.0.1:8000/
```

单图表地址：

```text
http://服务器IP:8000/chart?code=SH000001&lv=1m&days=60&source=mootdx
```

参数说明：

| 参数 | 示例 | 说明 |
| --- | --- | --- |
| `code` | `SH000001` / `000001.SZ` | 股票或指数代码 |
| `lv` | `1m` / `5m` / `15m` / `30m` / `60m` / `day` | K 线级别 |
| `days` | `60` | 拉取最近多少天数据 |
| `source` | `mootdx` / `eastmoney` | 行情数据源，默认通达信 |

## 依赖

服务端入口使用 Python 标准库 `http.server`，不需要 Flask/FastAPI。

建议使用 Python 3.10 或 3.11。当前代码已兼容 Python 3.10；如果服务器报 `cannot import name 'Self' from 'typing'`，说明部署目录还是旧代码，需要重新同步最新的 `Combiner/KLine_Combiner.py`、`Seg/Eigen.py`、`Seg/Seg.py`。

行情数据默认使用项目已有的通达信数据源 `DATA_SRC.MOOTDX`。需要确保运行环境已安装项目依赖，至少包含：

```bash
pip install -r Script/requirements.txt
pip install mootdx
```

## 部署注意

- 对外访问时必须使用 `--host 0.0.0.0`。
- 云服务器安全组或防火墙需要放行对应端口，例如 `8000`。
- 首页只是入口页，图表内容由 `/chart` 按请求实时生成，因此首次加载或切换代码时会等待行情接口返回。
- 当前服务不做鉴权；如果暴露到公网，建议放到 Nginx 后面并增加访问控制。
