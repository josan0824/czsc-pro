# 买卖点名词解释

本文整理项目中形态学买卖点（`bsp`）相关缩写、显示标签和核心判定逻辑，方便阅读图表与代码。

## 显示标签规则

图表上的买卖点标签由 `CBS_Point.desc()` 生成：

- `b` 表示买点，来自 `is_buy=True`。
- `s` 表示卖点，来自 `is_buy=False`。
- 后面的 `1`、`1p`、`2`、`2s`、`3a`、`3b` 是买卖点类型。
- 如果标签前有 `※`，表示这是线段级别买卖点（`segbsp`），不是普通笔级别买卖点。
- 如果同一笔同时满足多个类型，会用逗号合并，例如 `b2,3b`。

示例：

| 图上标签 | 含义 |
| --- | --- |
| `b1` | 一类买点 |
| `s1` | 一类卖点 |
| `b1p` | 盘整背驰一类买点 |
| `b2` | 二类买点 |
| `b2s` | 类二买点 |
| `b3a` | 三类买点，且中枢在一买之后形成 |
| `b3b` | 三类买点，且中枢在一买之前形成 |
| `s3b` | 三类卖点，且中枢在一卖之前形成 |
| `※b2` | 线段级别二类买点 |

相关代码：

- `Common/CEnum.py`：`BSP_TYPE` 定义了类型枚举。
- `BuySellPoint/BS_Point.py`：`type2str()` 和 `desc()` 定义显示文本。
- `BuySellPoint/BSPointList.py`：计算并登记各类买卖点。

## BSP_TYPE 类型

代码中的枚举与字符串如下：

| 枚举 | 字符串 | 名称 |
| --- | --- | --- |
| `BSP_TYPE.T1` | `1` | 一类买卖点 |
| `BSP_TYPE.T1P` | `1p` | 盘整背驰一类买卖点 |
| `BSP_TYPE.T2` | `2` | 二类买卖点 |
| `BSP_TYPE.T2S` | `2s` | 类二买卖点 |
| `BSP_TYPE.T3A` | `3a` | 三类买卖点，3-after |
| `BSP_TYPE.T3B` | `3b` | 三类买卖点，3-before |

`BSP_TYPE.main_type()` 返回字符串第一位，因此 `3a` 和 `3b` 的主类型都是 `3`。

## b / s 如何确定

`CBSPointList.add_bs()` 中用 `bi.is_down()` 判断买卖方向：

- 当前买卖点落在一笔下跌笔末端，则 `is_buy=True`，显示为 `b...`。
- 当前买卖点落在一笔上涨笔末端，则 `is_buy=False`，显示为 `s...`。

也就是说，`b2` 与 `s2` 用同一套二类逻辑，只是方向相反。

## 一类买卖点：1

一类买卖点对应 `BSP_TYPE.T1`，核心逻辑在 `CBSPointList.treat_bsp1()`。

判定要点：

- 当前线段已经有最后一个非一笔中枢。
- 离开中枢的一笔需要相对进入中枢的一笔形成背驰。
- 如果开启 `bs1_peak=True`，离开中枢的一笔还必须突破中枢内部相关极值。
- 满足 `min_zs_cnt` 和 `bsp1_only_multibi_zs` 等配置要求。

常见图表标签：

- `b1`：下跌离开中枢后背驰，形成一类买点。
- `s1`：上涨离开中枢后背驰，形成一类卖点。

## 盘整背驰一类买卖点：1p

`1p` 对应 `BSP_TYPE.T1P`，代码在 `CBSPointList.treat_pz_bsp1()`。

`p` 可理解为盘整背驰场景。它不是普通趋势离开中枢后的 `1`，而是在同一线段内用前后两笔比较：

- 当前笔与前两笔中的对应笔处于同一线段。
- 当前笔方向与线段方向一致。
- 当前笔要创新高或创新低。
- 当前笔相对前一同向笔的 MACD 指标满足背驰条件。

常见图表标签：

- `b1p`：盘整背驰一类买点。
- `s1p`：盘整背驰一类卖点。

## 二类买卖点：2

二类买卖点对应 `BSP_TYPE.T2`，代码在 `CBSPointList.treat_bsp2()`。

判定要点：

- 默认必须跟在一类买卖点之后，由 `bsp2_follow_1=True` 控制。
- 一类点之后，先有一笔反向突破笔 `break_bi`。
- 再后一笔回撤笔 `bsp2_bi` 的幅度不能超过 `max_bs2_rate * break_bi.amp()`。
- 满足条件后在 `bsp2_bi` 的末端登记二类买卖点。

常见图表标签：

- `b2`：一类买点后的首次回撤形成二类买点。
- `s2`：一类卖点后的首次反弹形成二类卖点。

## 类二买卖点：2s

`2s` 对应 `BSP_TYPE.T2S`，代码在 `CBSPointList.treat_bsp2s()`。

这里的 `s` 不是 sell，而是“类二”的后缀；买点会显示为 `b2s`，卖点会显示为 `s2s`。

判定要点：

- 它从二类买卖点之后继续向后找，每次跳过两笔，保持同向回试的节奏。
- 后续候选笔需要与前面的二类区域保持价格重叠。
- 不能破坏一类点后的突破笔，否则停止。
- 回撤比例仍受 `max_bs2_rate` 限制。
- `max_bsp2s_lv` 可以限制类二层级；`bsp2s_follow_2` 控制是否必须已有正式二类点。

常见图表标签：

- `b2s`：类二买点。
- `s2s`：类二卖点。

## 三类买卖点：3a

`3a` 对应 `BSP_TYPE.T3A`，代码在 `CBSPointList.treat_bsp3_after()`。

`a` 是 after，表示用于判定三类点的中枢出现在一类买卖点之后。

判定要点：

- 默认必须跟在一类买卖点之后，由 `bsp3_follow_1=True` 控制。
- 在一类点后的下一段走势中寻找第一个或前若干个非一笔中枢。
- 找到中枢离开笔之后的下一笔作为三类候选笔。
- 候选笔方向必须是对中枢的回试方向。
- 候选笔不能回到中枢区间内。
- 如果开启 `bsp3_peak=True`，候选笔还必须突破中枢涉及笔的极值。
- `bsp3a_max_zs_cnt` 控制最多检查一类点之后多少个中枢。

常见图表标签：

- `b3a`：向上离开一类买点后的新中枢，随后回试不回中枢，形成三类买点。
- `s3a`：向下离开一类卖点后的新中枢，随后反抽不回中枢，形成三类卖点。

## 三类买卖点：3b

`3b` 对应 `BSP_TYPE.T3B`，代码在 `CBSPointList.treat_bsp3_before()`。

`b` 是 before，表示用于判定三类点的中枢出现在一类买卖点之前，也就是一类点前最后一个非一笔中枢。

判定要点：

- 必须存在一类买卖点对应的 `bsp1_bi`。
- 取一类点所在段的最终非一笔中枢 `cmp_zs`。
- 从一类点之后隔两笔开始找三类候选笔。
- 候选笔不能回到 `cmp_zs` 的中枢区间内。
- 找到第一个满足条件的候选笔后登记 `3b` 并停止。
- 如果开启 `strict_bsp3=True`，要求该中枢的离开笔紧挨一类买卖点。

常见图表标签：

- `b3b`：一买之后回试一买之前的最后中枢，不回中枢，形成三类买点。
- `s3b`：一卖之后反抽一卖之前的最后中枢，不回中枢，形成三类卖点。

## 线段买卖点：segbsp

线段买卖点使用同一套 `CBSPointList` 计算，但输入对象从“笔列表”换成了“线段列表”。

配置在 `CChanConfig.set_bsp_config()` 中生成：

- `bs_point_conf`：普通笔级别买卖点配置。
- `seg_bs_point_conf`：线段买卖点配置。

线段买卖点在图上用 `※` 标识，例如：

- `※b1`
- `※b2`
- `※s3b`

默认线段买卖点使用 `macd_algo="slope"`，且 `bsp1_only_multibi_zs=False`。

## 相关配置项

| 配置项 | 作用 |
| --- | --- |
| `bs_type` | 关注哪些买卖点类型，默认 `"1,1p,2,2s,3a,3b"` |
| `divergence_rate` | 一类买卖点背驰阈值 |
| `min_zs_cnt` | 一类买卖点至少需要经历的中枢数量 |
| `bsp1_only_multibi_zs` | 统计一类点中枢数量时，是否只看至少三笔形成的中枢 |
| `max_bs2_rate` | 二类 / 类二回撤比例上限 |
| `bs1_peak` | 一类点是否必须突破相关极值 |
| `bsp2_follow_1` | 二类点是否必须跟在一类点之后 |
| `bsp3_follow_1` | 三类点是否必须跟在一类点之后 |
| `bsp3_peak` | 三类候选笔是否必须突破中枢极值 |
| `bsp2s_follow_2` | 类二点是否必须跟在正式二类点之后 |
| `max_bsp2s_lv` | 类二点最大层级 |
| `strict_bsp3` | 三类点对应中枢是否必须紧挨一类点 |
| `bsp3a_max_zs_cnt` | `3a` 最多检查一类点之后几个中枢 |

配置支持按买卖方向分别覆盖，后缀为：

- `xxx-buy`：只覆盖买点配置。
- `xxx-sell`：只覆盖卖点配置。

## 阅读图表时的建议

1. 先看标签前缀：`b` 是买点，`s` 是卖点。
2. 再看类型：`1/1p/2/2s/3a/3b` 表示买卖点类型。
3. 看到 `2s` 时不要误读为“二类卖点”，卖点会写成 `s2s`。
4. 看到 `3a/3b` 时重点看中枢位置：`3a` 看一类点之后的新中枢，`3b` 看一类点之前的最后中枢。
5. 看到 `※` 时说明这是线段级别信号，级别比普通笔级别更高。
